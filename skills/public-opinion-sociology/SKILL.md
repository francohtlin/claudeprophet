# Public Opinion And Sociology

Use this skill for polls, social behavior, adoption, protests, virality, public statements, and questions where population behavior matters.

## Evidence Sources

- polling data and methodology
- platform metrics if official or credible
- demographic/base-rate context
- institutional incentives
- historical analogues
- prediction markets as aggregated belief, if rule-equivalent

## Calibration Notes

- Treat social-media anecdotes as weak evidence unless backed by scale.
- Prefer base rates and measured surveys over vibes.
- Use uncertainty bands when the population or measurement source is ambiguous.
